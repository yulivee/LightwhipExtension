## Midi Channel conversion

### Red

| Ableton | Touch Designer | Effect     |
| ------- | -------------- | ---------- |
| A2      | `ch1n58`       | show       |
| G\# 2   | `ch1n57`       | pulse      |
| G2      | `ch1n56`       | hold       |
| F\# 2   | `ch1n55`       | slow_pulse |
| F2      | `ch1n54`       | ramp_out   |

Touch Designer:
```
# Channel names
ch1n55 ch1n56 ch1n57 ch1n58 ch1n54

# Rename to
slow_pulse hold show pulse ramp_out
```
